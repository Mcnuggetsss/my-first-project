# 食品批发销售管理系统 - 安装说明

## 方法一：复制到其他电脑（局域网使用）

### 步骤
1. 把整个 `sales_app` 文件夹复制到目标电脑（U盘/共享文件夹均可）
2. 目标电脑需要安装 Python 3：https://www.python.org/downloads/
   - 安装时勾选 **"Add Python to PATH"**
3. 双击 `启动.bat`，浏览器会自动打开

### 注意
- 每台电脑的数据库 `sales.db` 是独立的，数据不会自动同步
- 如果要共享数据，只需在一台电脑上运行，其他电脑通过局域网 IP 访问

---

## 方法二：上传到 GitHub（方便在多台电脑同步代码）

### 前提
- 注册 GitHub 账号：https://github.com
- 安装 Git：https://git-scm.com/download/win

### 步骤
打开命令行（在 sales_app 文件夹里），依次运行：
```
git init
git add .
git commit -m "初始版本"
git remote add origin https://github.com/你的用户名/sales_app.git
git push -u origin main
```

其他电脑克隆：
```
git clone https://github.com/你的用户名/sales_app.git
cd sales_app
pip install -r requirements.txt
双击启动.bat
```

---

## 方法三：部署到互联网（Railway）

### 步骤
1. 先完成"方法二"把代码上传到 GitHub
2. 注册 Railway：https://railway.app（用 GitHub 账号登录）
3. 点击 **New Project → Deploy from GitHub repo**
4. 选择你的 `sales_app` 仓库
5. Railway 会自动检测并部署，完成后给你一个公开网址

### 注意
- Railway 免费套餐每月有限额（约 500 小时）
- 数据库文件存在服务器上，**重新部署会清空数据**，建议定期备份 `sales.db`

---

## 备份数据

`sales.db` 文件就是全部数据，定期复制这个文件备份即可。
